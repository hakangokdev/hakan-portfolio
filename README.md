- 🌓 Dark/Light mode

- 🎨 Modern UI with smooth animations

- 🎯 SEO optimized
- 📝 Contact form

- React 18+
- TypeScript
- Tailwind CSS
- Framer Motion
- React Router DOM
- React Icons
- React Helmet Async

## 🛠️ Getting Started

### Prerequisites

- Node.js 16+ and npm

### Installation

1. Clone the repository
```bash
git clone https://github.com/hakangokdev/portfolio.git
cd portfolio
```

2. Install dependencies
```bash
npm install
```

3. Start the development server
```bash
npm run dev
```

4. Build for production
```bash
npm run build
```
